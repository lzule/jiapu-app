# KinCanvas Family Tree / 家谱可视化编辑器

中：离线家谱可视化编辑工具，支持拖拽建关系、分支折叠、版本管理、整谱矢量/超清导出。  
EN: Offline visual family tree editor with drag-and-link relationships, subtree collapse, version history, and full-tree SVG/UHD export.
